Можно еще раз и я думаю для всех пользователей, чтобы было понятно, мы:
1. Должны создать файл php.ini
2. Внутри его написать следующее:
<?php
phpinfo();
Expand text…
mail.add_x_header = On
mail.log = /адрес внутри сайта/public_html/phpmail.log
?>
3. Затем его положить внутри сайта или в папку etc/
4. В файле .htaccess добавить следующее:
<Files php.ini>
order allow,deny
deny from all
</Files>

Правильно или что-то не так написал?
