kazuho commented on 25 Sep 2015

Something is wrong with your configuration. H2O is known to work with host-named virtual hosting using TLS (and h2o.examp1e.net is configured as such).

Please refer to "Example. Using host-level listeners for HTTPS virtual-hosting" at https://h2o.examp1e.net/configure/base_directives.html#listen.

If you have difficulty, please provide a test case so that others could reproduce the issue.
