Install MySQL 5.6 on CentOS 7 / RHEL 7

MySQL is a popular open-source relational database management system widely used by a number of web platforms including Magento and WordPress. With the release of RHEL 7, MySQL is no longer included in the default RHEL / CentOS 7 repositories and has been replaced by MariaDB.

MariaDB is an excellent platform that is considered to be a drop-in replacement for MySQL, however, there are some features specific to MySQL 5.6 that have not yet been ported. At the time of this writing, the Magento  2.x platform specifically requires MySQL 5.6 for this reason.

If you find yourself needing MySQL 5.6 on a RHEL / CentOS 7 environment, have no fear, the installation is very simple and just requires a few short steps.

First, ensure that your system is completely up-to-date before we make any changes:

sudo yum update

 

1 – Install MySQL Community Repository

 

We recommend using the official MySQL community repository for the installation.

Download and install it using the following:

sudo rpm -Uvh http://repo.mysql.com/mysql-community-release-el7-5.noarch.rpm sudo rpm -ivh mysql-community-release-el7-5.noarch.rpm

sudo yum update

 

2 – Install MySQL

 

Now that the repository is installed, we can proceed to install MySQL (note that you may need to remove MariaDB if it was previously installed):

sudo yum install mysql-server

 

3 – Start MySQL and Enable on Boot

 

We can now proceed to start the newly installed MySQL service and enable it to start on boot:

sudo systemctl start mysqld

sudo systemctl enable mysqld

 

4 – Harden MySQL

 

It would now be a good idea to proceed to run the mysql_secure_installation script that addresses several security concerns in the default MySQL installation. You can read more about what this script addresses in the MySQL Reference Manual.

sudo mysql_secure_installation

 

5 – Finished!

 

Congratulations! You have now finished the installation of MySQL 5.6 and can now proceed to configure and use it for your applications such as Magento 2!
