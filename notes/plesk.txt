-------------------------------------------------------------
Решение:
-------------------------------------------------------------
Ответ:
После изучения настроек хостинг панели Plesk для Вашего сайта,
прояснилась ситуация с ошибкой "Ошибка установки соединения с базой данных".

Для возобновления работоспособности сайта необходимы следующие шаги:
1.Создать Базу Данных: inbloom.
2.Создать пользователя root и задать пароль.
3.Произвести резервное копирование базы данных для сайта с предидущего хостинга средствами панели управления.
4.Импортировать базу данных. Для этого можно воспользоваться статьей приведенной по ссылке далее
